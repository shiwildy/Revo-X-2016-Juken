# Revo X 2016 Juken

Repo ini berisi dokumentasi dan mapping untuk setup motor revo x 2016.

## Informasi Motor

- Tipe motor: Honda Revo FI / Revo X
- Tahun: 2016
- ECU: Juken 5++ Dualband
- Posisi mesin: 1 silinder, tegak miring (sekitar 80 derajat dari vertikal)

## Spesifikasi Dasar

- Mesin: 4-tak, SOHC, 2-klep, pendingin udara
- Bore x stroke: 50.000 x 55.597 mm
- Rasio kompresi: 9.3:1
- Sistem bahan bakar: PGM-FI
